# FlixSkip Website

Landing page for FlixSkip - a free browser extension that automatically skips Netflix intros, recaps, and "Are you still watching?" prompts.

## Live Site

https://nazeefmasood.github.io/flixskip-website/

## Features

- Responsive design optimized for all devices
- Dark Netflix-inspired theme
- SEO optimized with proper meta tags
- Schema.org structured data
- Fast loading with minimal dependencies

## Structure

```
flixskip-website/
├── index.html      # Main landing page
├── privacy.html    # Privacy policy
├── sitemap.xml     # XML sitemap for SEO
├── robots.txt      # Search engine directives
└── icons/          # Favicon and app icons
```
